## On-Page SEO Checklist

### Every Page Must Have:

- [ ] Unique title tag (50-60 characters)
- [ ] Meta description (150-160 characters)
- [ ] H1 heading with primary keyword
- [ ] H2-H3 subheadings with related keywords
- [ ] Keyword in first 100 words
- [ ] Internal links to related pages
- [ ] Image alt text with keywords
- [ ] URL slug includes keyword
- [ ] Mobile-friendly design
- [ ] Page load under 3 seconds

### Homepage Optimization:

```
Title: EyeView Sunglasses | OEM & Wholesale Sunglasses Manufacturer
Meta: Leading OEM sunglasses manufacturer. Custom logo, high quality, 
      competitive wholesale prices. MOQ 100pcs. Free samples. Get quote!
H1: Premium OEM Sunglasses Manufacturer & Wholesaler
Keywords in content: sunglasses wholesale, OEM sunglasses, custom sunglasses, 
                     bulk sunglasses, sunglasses manufacturer
```

## Technical SEO

### Site Speed Targets:
- First Contentful Paint: < 1.5s
- Largest Contentful Paint: < 2.5s
- Time to Interactive: < 3.5s
- Cumulative Layout Shift: < 0.1

### Schema Markup (Add to pages):

**Organization Schema (Homepage):**
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "EyeView Sunglasses",
  "url": "https://eyeviewsunglasses.com",
  "logo": "https://eyeviewsunglasses.com/logo.png",
  "description": "OEM sunglasses manufacturer and wholesaler",
  "foundingDate": "2010",
  "contactPoint": {
    "@type": "ContactPoint",
    "contactType": "sales",
    "email": "sales@eyeviewsunglasses.com"
  }
}
```

**Product Schema (Product pages):**
```json
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "Custom Sunglasses",
  "description": "OEM custom sunglasses with your logo",
  "brand": {
    "@type": "Brand",
    "name": "EyeView"
  },
  "offers": {
    "@type": "Offer",
    "priceCurrency": "USD",
    "minPrice": "8.00",
    "maxPrice": "25.00"
  }
}
```
