# 客户开发每日检查清单

## 每日任务 (30-60 分钟)

### LinkedIn (15 分钟)
- [ ] 发送 10-15 个连接请求
- [ ] 回复所有消息 (24 小时内)
- [ ] 查看推荐客户

### 邮件 (20 分钟)
- [ ] 发送 20-30 封开发邮件
- [ ] 回复所有询盘 (24 小时内)
- [ ] 追踪打开率和回复率

### B2B 平台 (5 分钟)
- [ ] 检查各平台询盘
- [ ] 刷新产品 listing

### 社交媒体 (10 分钟)
- [ ] 发布 1 条内容
- [ ] 回复评论/私信

### 数据录入 (10 分钟)
- [ ] 更新 CRM/追踪表
- [ ] 记录今日成果

## 每周任务

- [ ] 收集 50 个新客户邮箱
- [ ] 优化邮件模板
- [ ] 分析本周数据
- [ ] 生成周报

## 月度目标

- [ ] LinkedIn 新增 200 连接
- [ ] 发送 600-900 封邮件
- [ ] 获得 10-20 个询盘
- [ ] 成交 1-2 个客户
