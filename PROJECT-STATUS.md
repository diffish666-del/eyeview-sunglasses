   - Sign up free account

2. **Push code to GitHub**
   ```bash
   cd /home/admin/.openclaw/workspace/eyeview-sunglasses-site
   git init
   git add .
   git commit -m "Initial commit - EyeView website"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/eyeview-sunglasses.git
   git push -u origin main
   ```

3. **Deploy to Vercel**
   - Go to vercel.com
   - Sign in with GitHub
   - Import your repository
   - Click Deploy

4. **Connect Domain**
   - Add eyeviewsunglasses.com in Vercel
